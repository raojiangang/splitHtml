  "code": 0,

  "data": {
    "pageSize": 20,
    "totalPage": 1,
    "totalResult": 4,
    "currentPage": 1,
    "currentResult": 0,
    "list": [
        {

            "amount":"提现金额",
            "applyTime":"申请时间，13位时间戳",
            "payAmount":"打款金额",
            "payWay":"提现渠道，0：支付宝、1：微信",
            "payTime":"打款时间，13位时间戳",
            "status":"提现状态，0：提现中、1：成功、2：提现失败、3：个人撤销、4：审核中、5：审核通过",
            "statusDesc":"提现状态描述",
            "reason":"提现失败原因",
            "cancel":"是否允许撤销，0：不允许、1：允许",
            "delete":"是否允许删除，0：不允许、1：允许"
        }
    ]
  }
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
accessToken 是
uuid 是

Query

参数名称 是否必须 示例 备注
pageSize 是
currentPage 是

返回数据

名称类型是否必须默认值备注其他信息
codenumber必须
msgstring必须
dataobject必须
├─ pageSizenumber必须
├─ totalPagenumber必须
├─ totalResultnumber必须
├─ currentPagenumber必须
├─ currentResultnumber必须
├─ listobject []必须

item 类型: object

├─ idnumber必须
├─ amountnumber必须提现金额
├─ applyTimenumber必须申请时间
├─ payAmountnumber必须打款金额
├─ payWaynumber必须提现渠道，0：支付宝、1：微信
├─ payTimenumber必须打款时间
├─ statusnumber必须提现状态，0：提现中、1：成功、2：提现失败、3：个人撤销、4：审核中、5：审核通过
├─ statusDescstring必须提现状态描述
├─ reasonstring必须提现失败原因、提现成功备注，例：金额低于100扣除20%手续费等
├─ cancelnumber必须是否允许撤销，0：不允许、1：允许
├─ deletenumber必须是否允许删除，0：不允许、1：允许
